# TypeScript Project Template


## Description

Basic TypeScript template

## Getting Started

### Dependencies

* Install dependencies using `yarn` or `npm` 

```
yarn
```
```
npm run install
```

## Testing

### Run tests

* Run tests using `yarn` or `npm`
```
yarn test
```
```
npm run test
```

* Run tests using `yarn` or `npm` and watch mode
```
yarn test:watch
```
```
npm run test:watch
```
